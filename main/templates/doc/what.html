gae-init is everything you will need, to have a complete webapp with some of the coolest latest technologies, on Google App Engine.

If you have some experience already, building web applications, you are aware of the fact that there are some common tasks you have to do before you get started. You have to set up a project with a file structure, grab the latest libraries, copy usefull code snippets from old projects, or rewrite them, setup a login mechanism and other repeated boring stuff. gae-init solves exactly that problem. It comes with the most common stuff that will need to setup your modern web application. No more wasted time on dealing with tools and configurations. You can start working on your next big thing, right away!

gae-init is always up to date with the latest libraries and following the best programming practices. You like it clean? We love it! We religiously follow coding conventions for clarity and consistency. There are constant reviews on the codebase in order to provide for ourselves in first place sanity. Check out the full feature list, get started with the tutorial or simply fork it.

This project wouldn't be here without the my old fellas, Mr. Tim and Mr. Chris that contributed with ideas, code, RedBulls, other stuff and the pretty awesome logo that was made by Mr. Kostas. Some may ask why not using something that is already there like gae-boilerplate or flask-appengine-template. The answer is that, among other many things, gae-boilerplate's base.html is huge, unreadable and a bit chaotic compare to gae-init's base.html and that flask-appengine-template doesn't even have the pretty useful user sign-in.
