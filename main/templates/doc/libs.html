Gae-init provides a very easy and flexible way for integrating libraries into your project.

There are two type of extra libraries you can include in your projects. For every type of library we will describe follows an example of it being used in the project.

Client JavaScript

Do you want to include in your project a new fancy JavaScript media player? Download the desired lib in a temporary directory.

JQuery is an example.

Server Python

The libraries you can use server side are separated in two categories.

Supported by AppEngine

You can have a look at AppEngine's third party libraries support. Jinja2 is an example of that.

Extra Python Libraries advanced

Flask is an example of that.
