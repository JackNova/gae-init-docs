Clone the gae-init project to get the code locally on your computer

$ hg clone https://lipis@bitbucket.org/lipis/gae-init phonebook

This will create a new directory that is called phonebook and inside that directory there is another directory called main that containes all the source files, libraries and build scripts.

Change your current directory to phonebook/main and run the command npm install to install locally all the nessesery dependencies that are needed in the next step. 

$ cd phonebook/main
$ npm install
