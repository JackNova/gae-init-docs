Clone the gae-init project to get the code locally on your computer using git

$ git clone https://github.com/gae-init/gae-init.git phonebook

If you are more familiar with mercurial then use that

$ hg clone https://bitbucket.org/gae-init/gae-init phonebook

This will create a new directory that is called phonebook and inside that directory there is another directory called main that contains all the source files, libraries and more.
