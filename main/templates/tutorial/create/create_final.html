Finalise Contact Creation

Before we can actually test the contact creation there are couple of things that we have to complete.

Import contact.py

We will have to import the contact.py in the main.py, because otherwise the Flask application won't be able to figure out the routing rules. Inlcude the following line bellow the rest of the imports in the main.py file. 

import contact

Adding a link on the top bar

The most important thing to make the user experience better, we will have to create a link for the users to be able to start adding new contacts in their phonebook.

Add the lines 2 - 4 inside the <ul class="nav">...</ul> element that you will find in the top_bar.html file that is located in the templates/bit directory. 

<ul class="nav">
  <li class="{% if html_class == 'contact-create' %} active {% endif %}">
    <a href="{{url_for('contact_create')}}"><i class="icon-file"></i> Create Contact</a>
  </li>
  ...
</ul>
