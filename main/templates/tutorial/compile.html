Since gae-init is using CoffeeScript for writting scripts and LESS for writting stylesheets it is nessesary to compile the sources files into *.js and *.css respectivly. Also it is nessesery to zip all the third party Python libraries that are being used before running the application, because this makes the deployment of your app faster and smaller in number of files.

Run the build.py script from the main directory by providing the the -c argument, which stands for a clean build. 

$ ./build.py -c
