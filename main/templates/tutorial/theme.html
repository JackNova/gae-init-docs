Since gae-init uses Bootstrap there are plenty sites that are offering free and paid templates for it. Our personal favorite place to start is the Bootswatch.com. For this example we are going to use the Flatly but feel free to choose whatever you want.

Bootswatch is offering two options for downloading the themes: in pure CSS format or in LESS format. Since we are supporting LESS files is preferable to download the two LESS files that you can find for each of the themes.

In order to experiment with more themes we will create a new directory called theme under the directory main/static/src/style. Inside this new directory for every theme that we would like to test, we will create another directory with it's own name.

Follow the steps below for any theme that you would like to test from the Bootswatch's collection:

  1. Create a new directory called flatly
  2. Visit the Flatly
  3. Download the variables.less and bootswatch.less
  4. Put them in the new dicrectory flatly under main/static/src/style/theme

After following these steps your directory structure should look like this: 

static
    src
        style
            base.less
            theme
                flatly
                    bootswatch.less
                    variables.less

Open the style.less that is located in the main/static/src/less and add the following lines before the @import base 

...
// Theme
@import "theme/flatly/bootswatch";
@import "theme/flatly/variables";
...

To test it locally you will have to compile the LESS files into CSS and this is achieved by using the run.py script. Execute the script from the root directory like this (or you could provide the -w to monitor the LESS files for changes and will be compiled on save): 

$ ./run.py -c

If the script compiled without any error then you should see the new theme by refreshing your browser or by visiting: http://localhost:8080
