We assume that you are already know how to run the Google App Engine applications from a command line, but if not you should refer to the documentation .

Execute in your terminal the run.py from the root directory 

$ cd /path/to/phonebook
$ ./run.py -s

If everything went smoothly you can test the application by visiting the following URL in your web browser:

http://localhost:8080/

Feel free to spend some time by clickng on the different menus. As a first change withing the app you can signin in as adminstrator and after visiting the http://localhost:8080/admin/config/ change the Brand Name from gae-init to Phone Book (or to anything else you like). By doing that you should see on the top left corner the new name for your application.
