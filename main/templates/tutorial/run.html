We assume that you are already know how to run the Google App Engine applications from a command line, but if not you should refer to the documentation .

Preferably in another terminal window run the dev_appserver.py 

$ dev_appserver.py phonebook/main

Or you can run the command dev_appserver.py . within the main directory. 

$ cd phonebook/main
$ dev_appserver.py .

If everything went smoothly you can test the application by visiting the following URL in your web browser:

http://localhost:8080/

Feel free to spend some time by clickng on the different menus. As a first change withing the app you can signin in as adminstrator and after visiting the http://localhost:8080/admin/config/ change the Brand Name to Phonebook (or to anything else you like). If everything went OK, on the top left corner you should see the new name for your application.
