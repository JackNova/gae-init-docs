Generates a link that will add (or replace) &property=value to the current request or will remove it it's already there. Optionally instead of showing the actual value it's possible to show an icon which is usually good for filtering boolean properties.

Arguments
property
The name of the entity's property.
value
The value to be applied to the filter.
icon
Optional name for the icon instead that will be shown instead of the value.

Examples & Demo

Check how it is used in user_list.html and play with filters in User List.
