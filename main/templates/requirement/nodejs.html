In order to compile CoffeeScript files into JavaScript, LESS files into CSS and then minify all the JavaScript dependences, gae-init is using CoffeeScript, LESS and UglifyJS compilers respectivly.

Download the latest node.js and install it globally on your system. This installation also includes the Node Packaged Modules which is a very easy way to install packages.

After the installation make sure that you are able to run the npm command from your terminal.
